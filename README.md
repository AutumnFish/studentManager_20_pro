# studentManager_20
学员管理 基于 express mongoDB bootstrap

## 包
1. express 让我们能够通过浏览器访问 托管静态资源
    cnpm install express --save


## 项目基本结构

```
    node_modules ->第三方包
    static ->静态文件
    .gitignore ->忽略文件
    app.js ->入口文件
    package.json ->项目配置文件
    README.md ->github的项目说明
```

## 步骤记录

1. github托管
    github建库 选择 忽略 node的文件
    git clone 到本地
    npm init -y 初始化为项目

2. 资源导入
    bootstrap

3. 静态页面实现
